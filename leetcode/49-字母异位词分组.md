# 49. 字母异位词分组
> [LeetCode题目链接](https://leetcode.cn/problems/group-anagrams/?envType=study-plan-v2&envId=top-interview-150)

## 排序后构建哈希表
```cpp
class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        // 采用哈希表来维护满足题设的分组
        std::unordered_map<std::string, std::vector<std::string>> mp;
        // key - 排序后的字符串, value - vector[原始字母字符]
        for (const auto& str : strs) {
            std::string sort_key = str;
            std::sort(sort_key.begin(), sort_key.end());
            mp[sort_key].push_back(str);
        }
        // 建立vector<vector<string>>来返回参数
        vector<vector<string>> ans;
        for (auto it = mp.begin(); it != mp.end(); ++it) {
            ans.push_back(it->second);
        }
        return ans;
    }
};
```
