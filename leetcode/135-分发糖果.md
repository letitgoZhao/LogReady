# 135. 分发糖果


> [题目链接](https://leetcode.cn/problems/candy/solutions/533150/fen-fa-tang-guo-by-leetcode-solution-f01p/?envType=study-plan-v2&envId=top-interview-150)


n 个孩子站成一排。给你一个整数数组 ratings 表示每个孩子的评分。

你需要按照以下要求，给这些孩子分发糖果：
- 每个孩子至少分配到 1 个糖果。
- 相邻两个孩子评分更高的孩子会获得更多的糖果。
- 请你给每个孩子分发糖果，计算并返回需要准备的 最少糖果数目 。

## 正序、倒序遍历
时间复杂度`o(n)`, 空间复杂度`o(n)`。
```cpp
class Solution {
public:
    int candy(vector<int>& ratings) {
        // '相邻的孩子中, 评分高的孩子获得更多的糖果'
        // 左规则: 当ratings[i - 1] < ratings[i]时, i号多
        // 右规则: 当ratings[i] > ratings[i + 1]时, i号多
        // 方式一, 遍历ratings数组两次, 分别计算满足左规则和右规则的最少糖果数
        int n = ratings.size();
        vector<int> candidate(n, 1);
        for (int i = 1; i < n; ++i) { // 左规则
            if (ratings[i] > ratings[i - 1]) {
                candidate[i] = candidate[i - 1] + 1;
            }
        }
        for (int i = n - 2; i >= 0; i--) { // 右规则
            if (ratings[i] > ratings[i + 1]) {
                // 需要满足左规则和右规则
                candidate[i] = max(candidate[i], candidate[i + 1] + 1);
            }
        }
        return std::accumulate(candidate.begin(), candidate.end(), 0);
    }
};
```
