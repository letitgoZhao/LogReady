# 155. 最小栈

> [题目链接](https://leetcode.cn/problems/min-stack/?envType=study-plan-v2&envId=top-interview-150)

## 栈结构操作

```cpp
class MinStack {
private:
    // 常数时间内检索到最小元素
    // 实现策略:
    // 1. 使用STL-stack来实现栈结构
    // 2. 初始状态下, 没有元素, 最小元素初始为无穷大
    // 3. 每次push时都要更新最小元素, 实现o(1)检索
    // 4. 每次pop时也要丢弃最小元素, 表示丢弃上一个状态
    stack<int> elem_stack;
    stack<int> min_stack;

public:
    MinStack() { min_stack.push(INT_MAX); }

    void push(int val) {
        elem_stack.push(val);
        min_stack.push(min(val, min_stack.top()));
    }

    void pop() {
        elem_stack.pop();
        min_stack.pop();
    }

    int top() { return elem_stack.top(); }

    int getMin() { return min_stack.top(); }
};
```
