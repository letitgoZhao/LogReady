# 80. 删除有序数组中的重复项II

> [题目链接](https://leetcode.cn/problems/remove-duplicates-from-sorted-array-ii/?envType=study-plan-v2&envId=top-interview-150)

## 双指针检索+重复情况判别

```cpp
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int sz = nums.size();
        int slow = 0, fast = 0, len_new = 1;

        while (fast < sz) {
            fast = slow + 1;
            // 阻塞迭代fast
            while (fast < sz && nums[fast] == nums[slow]) {
                fast++;
            }
            // 说明有两个及以上的重复
            if (fast - slow >= 2) {
                nums[++len_new - 1] = nums[slow];
                slow = fast - 1;
            }
            // 处理结束标志, 如不处理后续抛出异常, 相当于已经获知结束
            if (fast == sz) {
                break;
            }
            // 处理不同元素情况
            nums[++len_new - 1] = nums[fast];
            slow++;
        }
        return len_new;
    }
};
```
