# 20. 有效的括号

> [题目链接](https://leetcode.cn/problems/valid-parentheses/description/?envType=study-plan-v2&envId=top-interview-150)

## 栈结构+unordered_map结构+API调用

```cpp
class Solution {
public:
    bool isValid(string s) {
        int n = s.size();
        if (n % 2 == 1)
            return false; // 排除简单情况

        unordered_map<char, char> pairs = {{')', '('}, {']', '['}, {'}', '{'}};
        stack<char> st;

        for (auto ch : s) {
            // 检测键值是否存在, 来决定出栈, 并判断是否满足题设
            if (pairs.count(ch)) {
                // 栈为空直接返回结果, 不执行st.top(), 防止报错, 注意逻辑顺序
                if (st.empty() || st.top() != pairs[ch])
                    return false;
                // 栈顶内容为( [ { 并且检测为)、}、]即配对
                st.pop();
            } else
                st.push(ch); // 压栈
        }
        return st.empty();
    }
};
```
