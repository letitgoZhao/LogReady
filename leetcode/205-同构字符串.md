# 205. 同构字符串

> [LeetCode题目链接](https://leetcode.cn/problems/isomorphic-strings/description/?envType=study-plan-v2&envId=top-interview-150)

## unordered_map哈希表检测

> 考察“双射关系”，因而维护两个哈希表，对两个表进行检测。
> (保证不重复增加映射关系；同时双表才能保证“双射关系”，在if - condition体现)。

```cpp
class Solution {
public:
    bool isIsomorphic(string s, string t) {
        if (s.size() != t.size())
            return false;
        // 因为双射的存在, 建立两个哈希表来维护这种映射关系
        unordered_map<char, char> s2t, t2s;
        // s的任意一个字符被t唯一对应, 同时t的任意一个字符被s唯一对应
        // s2t: key-s, val - t, <a, b>
        // t2s: key-t, val - s, <b, a>
        for (int i = 0; i < s.size(); ++i) {
            char a = s[i], b = t[i];
            // a 或 b 是已有的键值, 但映射关系不匹配
            if ((s2t.find(a) != s2t.end() && s2t[a] != b) ||
                (t2s.find(b) != s2t.end() && t2s[b] != a))
                return false; // 为什么 ||, 因为s2t的键值可能没有a元素,
                              // 但a为s2t的映射
            // 新添加 映射关系
            s2t[a] = b;
            t2s[b] = a;
        }
        return true;
    }
};
```
