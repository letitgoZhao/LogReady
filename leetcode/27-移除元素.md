# 27. 移除元素

> [题目链接](https://leetcode.cn/problems/remove-element/?envType=study-plan-v2&envId=top-interview-150)

## 初次尝试——遍历搜索

```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int len_new = 0;
        int sz = nums.size();
        for (int i = 0; i < sz; ++i) {
            if (nums[i] != val) {
                nums[len_new++] = nums[i];
            }
        }
        return len_new;
    }
};
```
