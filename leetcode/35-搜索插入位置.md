# 35.搜素插入位置

> [题目链接](https://leetcode.cn/problems/search-insert-position/description/?envType=study-plan-v2&envId=top-interview-150)

## 二分查找算法(闭区间写法, 注意下标)
```cpp
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        // nums已经为升序排序, o(lgn)的检索方案, 考虑二分查找方案
        int l = 0, r = nums.size() - 1;

        while (l <= r) {
            // 防止加法运算移除, 同时提高程序的运行速度
            int mid = l + ((r - l) >> 1);
            if (nums[mid] < target)
                l = mid + 1;
            else 
                r = mid - 1;
        }
        return l;   // 或者right + 1
    }
};
```
