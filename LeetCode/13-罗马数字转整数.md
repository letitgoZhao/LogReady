# 13. 罗马数字转整数

> [题目链接](https://leetcode.cn/problems/roman-to-integer/?envType=study-plan-v2&envId=top-interview-150)

## 模拟

```cpp
class Solution {
public:
    int romanToInt(string s) {
        unordered_map<char, int> symbol = {{'I', 1},   {'V', 5},   {'X', 10},
                                           {'L', 50},  {'C', 100}, {'D', 500},
                                           {'M', 1000}};
        int ans = 0;
        for (int i = 0; i < s.size(); ++i) {
            int value = symbol[s[i]];
            if (i < s.size() - 1 && value < symbol[s[i + 1]])
                // 特殊情况减, 如IV = -1 + 5 = 4
                ans -= value;
            else
                // 正常情况直接加, 如 III = 1 + 1 + 1 = 3
                ans += value;
        }
        return ans;
    }
};
```
