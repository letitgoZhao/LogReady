# 242. 有效的字母异位词

> [题目链接](https://leetcode.cn/problems/valid-anagram/?envType=study-plan-v2&envId=top-interview-150)


## std::sort方法使用——题目规律发现

```cpp
class Solution {
public:
    bool isAnagram(std::string s, std::string t) {
        if (s.size() != t.size())
            return false;
        // 若为有效的字母异位词, 则排序后string一致
        std::sort(s.begin(), s.end());
        std::sort(t.begin(), t.end());

        return (s == t);
    }
};
```

## vector构建'哈希表'——题目要求

```cpp
class Solution {
public:
    bool isAnagram(std::string s, std::string t) {
        if (s.size() != t.size()) {
            return false;
        }
        vector<int> ans_table(26, 0);
        for (const auto& ch : s) {
            ans_table[ch - 'a']++;  // 生成表和对应的字母次数
        }

        for (const auto& ch : t) {
            ans_table[ch - 'a']--;  // 验证表
            if (ans_table[ch - 'a'] < 0)
                return false;
        }
        // 验证表中的元素在 '生成表' 中均有且出现的次数一致
        // 由于生成表和 '验证表' 的 '长度'(遍历次数) 一致, 因而两者 '等价'(满足题设)
        return true;
    }
};
```
