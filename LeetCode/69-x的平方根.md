# 69. x的平方根

> [题目链接](https://leetcode.cn/problems/sqrtx/?envType=study-plan-v2&envId=top-interview-150)


## 计算器算法
指数函数、对数函数的使用。

```cpp
class Solution {
public:
    int mySqrt(int x) {
        if (x == 0)
            return 0;
        // e^(0.5 * ln x) = sqrt(x), float -> int
        int ans = std::exp(0.5 * log(x));
        // 注意long long
        return (long long)(ans + 1) * (ans + 1) <= x ? ans + 1 : ans;
    }
};
```

## 二分查找算法
由于 x 平方根的整数部分 ans 是满足 $k^2<=x$ 的最大 k 值，因此我们可以对 k 进行二分查找，从而得到答案。

```cpp
class Solution {
public:
    int mySqrt(int x) {
        int l = 0, r = x, ans = -1;
        while (l <= r) {
            int mid = l + ((r - l) >> 1);
            if ((long long)mid * mid <= x) {
                l = mid + 1;
                ans = mid;
            } else {
                r = mid - 1;
            }
        }
        return ans;
    }
};
```
