# 637. 二叉树的层平均值

> [题目链接](https://leetcode.cn/problems/average-of-levels-in-binary-tree/description/?envType=study-plan-v2&envId=top-interview-150)

**题目要求:**

给定一个非空二叉树的根节点 `root` , 以数组的形式返回每一层节点的平均值。与实际答案相差 `10-5` 以内的答案可以被接受。注意: 树中节点数量在`[1, 104]`范围内, 且`-231 <= Node.val <= 231 - 1`。

```txt
输入数据: root = [3,9,20,null,null,15,7]。
输出数据: [3.00000,14.50000,11.00000]。
```

**方案实现:**

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
from collections import deque
class Solution:
    def averageOfLevels(self, root: Optional[TreeNode]) -> List[float]:
        # 二叉树的层次遍历
        if root is None:
            return []
        # 初始化队列结构
        q = deque()
        q.append(root)
        ans_list: List[float] = []

        while q:
            level_sz = len(q)
            level_list: List[int] = []
            for i in range(level_sz):
                # 获取当前节点的数据
                node = q.popleft()
                # 依次添加层元素数据
                level_list.append(node.val)
                # 判断并执行是否更新node的下一层数据
                if node.left is not None:
                    q.append(node.left)
                if node.right is not None:
                    q.append(node.right)
            # 当前层处理完, 执行特定的任务
            ans_list.append(
                sum(level_list) / len(level_list)
            )
        return ans_list
```
