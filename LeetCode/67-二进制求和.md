# 67. 二进制求和

> [题目链接](https://leetcode.cn/problems/add-binary/description/?envType=study-plan-v2&envId=top-interview-150)

给你两个二进制字符串 `a` 和 `b`，以二进制字符串的形式返回它们的和。

输入:`a = "11"`, `b = "1"`; 输出：`"100"`。

## 解答思路

初次尝试结果, 技术路线：`反转字符串, 字符串操作, 字符串的拼接, 时刻考虑进位问题, str <-> int的问题`。


```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        len_a = len(a)
        len_b = len(b)
        cur_res = ''
        
        # 反转操作需要
        a = a[::-1]
        b = b[::-1]

        iter_num, up = 0, 0
        
        # 处理两个字符串的相同部分
        while iter_num < len_a and iter_num < len_b:
            temp = int(a[iter_num]) + int(b[iter_num]) + up  # 当前位加上进位
            mod = temp % 2  # 取当前位
            up = temp // 2  # 进位
            cur_res = str(mod) + cur_res  # 拼接结果
            iter_num += 1

        # 处理长串a
        while iter_num < len_a:
            temp = int(a[iter_num]) + up
            mod = temp % 2
            up = temp // 2
            cur_res = str(mod) + cur_res
            iter_num += 1

        # 处理长串b
        while iter_num < len_b:
            temp = int(b[iter_num]) + up
            mod = temp % 2
            up = temp // 2
            cur_res = str(mod) + cur_res
            iter_num += 1

        # 如果还有进位，追加1
        if up == 1:
            cur_res = '1' + cur_res

        return cur_res
```

直接采用Python内置的API进行操作, 主要涉及到`bin`函数的使用, 以及索引高效检索的使用。

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        # 将二进制字符串转换为整数
        num_a = int(a, 2)
        num_b = int(b, 2)
        
        # 进行加法运算
        result = num_a + num_b
        
        # 将结果转换为二进制字符串，去掉 '0b' 前缀
        return bin(result)[2:]
```
