# 1. 两数之和

> [题目链接](https://leetcode.cn/problems/two-sum/?envType=study-plan-v2&envId=top-interview-150)

## 暴力求解 
```cpp
class Solution {
public:
    std::vector<int> twoSum(std::vector<int>& nums, int target);
};

std::vector<int> Solution::twoSum(std::vector<int>& nums, int target) {
    /* 题目要求简化了 - 说明 只会存在一个有效的答案 */
    for (int i = 0; i < nums.size() - 1; ++i) {
        for (int j = i + 1; j < nums.size(); ++j) {
            if (nums[i] + nums[j] == target) {
                return {i, j};
            }
        }
    }
    return {};
}
```

## unordered_map数据结构使用-hashtable常量级检索方案

```cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        /* hashtable-键值对来进行内循环的检索优化 */
        std::unordered_map<int, int> hashtable;
        for (int var = 0; var < nums.size(); var++) {
            const auto it = hashtable.find(target-nums[var]);
            if (it != hashtable.end())
                return {it->second, var}; // 找到了
            hashtable[nums[var]] = var; // (key,value) -> (nums[i], i), 保证不重复
        }
        return {};  // 没找到
    }
};
```
