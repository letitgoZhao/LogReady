# 141. 环形链表

> [题目链接](https://leetcode.cn/problems/linked-list-cycle/?envType=study-plan-v2&envId=top-interview-150)

## 初次尝试——快慢指针方式

假想「乌龟」和「兔子」在链表上移动，「兔子」跑得快，「乌龟」跑得慢。当「乌龟」和「兔子」从链表上的同一个节点开始移动时，如果该链表中没有环，那么「兔子」将一直处于「乌龟」的前方；如果该链表中有环，那么「兔子」会先于「乌龟」进入环，并且一直在环内移动。等到「乌龟」进入环时，由于「兔子」的速度快，它一定会在某个时刻与乌龟相遇，即套了「乌龟」若干圈。

我们可以根据上述思路来解决本题。具体地，我们定义两个指针，一快一慢。慢指针每次只移动一步，而快指针每次移动两步。初始时，慢指针在位置 head，而快指针在位置 head.next。这样一来，如果在移动的过程中，快指针反过来追上慢指针，就说明该链表为环形链表。否则快指针将到达链表尾部，该链表不为环形链表。


```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    bool hasCycle(ListNode* head) {
        //  处理极端情况
        if (head == nullptr || head->next == nullptr)
            return false;
        // 处理正常情况 - 链表是否有环(链表至少有两个元素)
        ListNode *fast_ = head->next, *short_ = head;
        while (fast_ != nullptr && fast_->next != nullptr &&
               short_ != nullptr) { // 注意逻辑顺序
            if (fast_ == short_)
                return true; // 链表有环, 地址相同
            // fast走两步, short走一步
            fast_ = fast_->next->next;
            short_ = short_->next;
        }
        return false;
    }
};
```

## 哈希集解决方案

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    bool hasCycle(ListNode *head) {
        std::unordered_set<ListNode*> hash;
        while (head != nullptr) {
            if (hash.count(head)) {
                // 查看地址是否重复
                return true;
            }
            // 增加元素
            hash.emplace(head);
            head = head->next;
        }
        return false;
    }
};
```
