# 26. 删除有序数组中的重复元素

> [题目链接](https://leetcode.cn/problems/remove-duplicates-from-sorted-array/?envType=study-plan-v2&envId=top-interview-150)

## 初次尝试——双下表检索

```cpp
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int sz = nums.size();
        int slow = 0, fast = 0, len_new = 1;
        while (fast < sz) {
            fast = slow + 1;
            while (fast < sz && nums[fast] == nums[slow])
                fast++;
            if (fast == sz)
                break;
            // 这里要注意数组索引和数组长度的区别
            nums[++len_new - 1] = nums[fast];
            slow = fast;
        }
        return len_new;
    }
};
```
