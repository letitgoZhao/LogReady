# 392. 判断子序列

> [题目链接](https://leetcode.cn/problems/is-subsequence/?envType=study-plan-v2&envId=top-interview-150)

## 双下标暴力匹配

时间复杂度：O(n+m)，其中 n 为 s 的长度，m 为 t 的长度。每次无论是匹配成功还是失败，都有至少一个指针发生右移，两指针能够位移的总距离为 n+m。
空间复杂度：O(1)。
```cpp
class Solution {
public:
    bool isSubsequence(string s, string t) {
        int subLen = s.size(), srcLen = t.size();
        if (subLen > srcLen)
            return false;
        if (subLen == srcLen)
            return (s == t);

        int i1 = 0, i2 = 0;
        while (i1 < subLen && i2 < srcLen) {
            if (s[i1] == t[i2])
                i1++;
            i2++;
        }
        return i1 == subLen;
    }
};
```

## 队列结构解决

```cpp
class Solution {
public:
    bool isSubsequence(string s, string t) {
        queue<char> q;
        // 队列结构
        for (const auto& ch : s)
            q.push(ch);
        // 从t字符串迭代查询
        for (const auto& ch : t)
            if (ch == q.front())
                q.pop();
        
        return q.empty();
    }
};
```
